Info

Clonezilla is software that can create images of partitions or whole disks for replication or backups. I've used it a lot over the years and it has saved me many times. While functionally excellent, Clonezilla uses an ncurses text based UI. I thought it would be interesting to intentionally put a console UI within a web context.
